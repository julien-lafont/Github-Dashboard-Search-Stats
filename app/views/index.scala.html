About Github Dashboard

Github Dashboard is a web application to search for git repositories on GitHub and to view information and statistics about these projects.
Server
PlayFramework 2.0 with Scala and Sindi (IOC container).
Client
Backbone (MVC framework), jQuery (dom, Ajax), Lawnchair (localstorage), Handlebars (templating) and UnderscoreJS (utility-belt).
Test
Spec2 & Selenium

Changelog

14/04/2012
Better exception handling (api return error detail in json)
Adapt backbone router to manage these errors
Add functionals tests (Selenium)
Improve code comments

Public API

This server is a middleware between Github, Yahoo and other WS. It normalizes data and provides a sustainable API.
