Finder API

Repository search

Search github repositories by keywords and main language.

/api/v1/repository/search/query
/api/v1/repository/search/query/page
/api/v1/repository/search/query/page/language

Query
Search these keywords (tip: you can use username:xx, fork:true, language:yy)
Page
Optional Page number
Language
Optional Main language

Repository API

Details

Retreive full information for a repository

/api/v1/repository/user/repo/detail

User
Owner of the repository
Repo
Name of the repository

Commits

Retreive last commits for a repository

/api/v1/repository/user/repo/commits /api/v1/repository/user/repo/commits/nb /api/v1/repository/user/repo/commits/nb/sha

Nb
Optional Number of results
Sha
Optional Sha of first commit

Contributors

Fetch all contributors (= commiters)

/api/v1/repository/user/repo/contributors

Watchers

Fetch all watchers

/api/v1/repository/user/repo/watchers

Languages

Fetch languages used in the repository

/api/v1/repository/user/repo/languages

User API

Details

Retreive full information about a commiter

/api/v1/user/user/detail

Repositories

List user's repositories

/api/v1/user/user/repositories

Geopositioning

Retreive location info about this user : Country, city, geo-positions
Only if the "location" field isn't empty !

/api/v1/user/user/geolocalisation

Stats API

Languages

List languages used by this user (name and percentage)

/api/v1/stats/user/languages

Impact

Retreive impact of each contributor on a repository (% of commits)

/api/v1/stats/user/repo/impact

Timeline

Retreive timeline projection of commits (number of commits per day)

/api/v1/stats/user/repo/timeline

}
